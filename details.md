# 💰 AI Money Mentor — Personal Finance Copilot for India

## 📌 Problem Statement

95% of Indians do not have a structured financial plan. Financial advisors charge ₹25,000+ annually and are primarily accessible to High Net-Worth Individuals (HNIs).  

This project aims to build an **AI-powered personal finance mentor** that democratizes financial planning — making it as simple and accessible as using WhatsApp.

---

## 🎯 Objective

To design and develop an intelligent financial assistant that:
- Understands user financial behavior
- Provides actionable financial insights
- Builds personalized financial plans
- Automates financial decision-making

---

## 🚀 Key Features

### 1️⃣ Money Health Score
A comprehensive financial wellness evaluation system.

#### Inputs:
- Monthly income
- Expenses
- Savings
- Loans
- Investments

#### Output:
- Score (0–100)
- Breakdown:
  - Emergency preparedness
  - Debt health
  - Investment diversification
  - Tax efficiency
  - Retirement readiness
  - Insurance coverage

## ⚙️ Core Financial Logic

### Savings Calculation
```

Savings = Income - Expenses
Savings Ratio = Savings / Income

```

### Emergency Fund
```

Emergency Fund = 6 × Monthly Expenses

```

### SIP Calculation
```

Monthly SIP = Goal Amount / (Years × 12)

```
